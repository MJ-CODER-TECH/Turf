# TurfZone Admin Panel

A modern admin dashboard for TurfZone turf booking platform.

## Tech Stack
- **React 18** + **Vite**
- **Tailwind CSS v3**
- **Recharts** for analytics
- **Lucide React** for icons
- **Outfit + Syne** fonts (Google Fonts)

## Features
- 📊 Dashboard with live stats & charts
- 📅 Bookings Management (filter, search, confirm/cancel)
- 🏟️ Turfs Management (grid & list view, add/edit/delete)
- 👥 Users Management (search, filter, ban)
- 💳 Payments & Revenue tracking
- 🕐 Time Slots Management (block/unblock slots)
- 📈 Reports & Analytics (charts, KPIs, top turfs)
- ⚙️ Settings (general, notifications, security)

## Setup

```bash
# Install dependencies
npm install

# Start dev server
npm run dev

# Build for production
npm run build
```
